## Project Overview
This project analyzes food delivery sales data using SQL and Tableau.
The objective is to clean raw data, design a star schema, compute KPIs,
and generate business insights.

## Tech Stack
- MySQL
- SQL (CTEs, Window Functions)
- Tableau
- Data Modeling

## Key Features
- Data cleaning & deduplication
- Star schema (fact & dimensions)
- KPI & trend analysis
- Interactive dashboards

## Business Insights
- City-wise demand patterns
- Cuisine & restaurant performance
- Customer spending behavior
